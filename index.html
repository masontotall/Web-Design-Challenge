Summary: City Latitude vs. Climate

This website is here to show you an analysis of how weather changes as you approach the equator. We pulled data from the OpenWeatherMap API and used a dataset of over 500 cities.

Using Matplotlib, we were able to plot different weather conditions and how they changed based on latitude. We looked at temperature, humidity, cloudiness, and wind speed. You can veiw the data visualizations and descriptions throughout this site.
